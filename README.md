# 🌐 Personal Portfolio Website

A modern, responsive **Personal Portfolio Website** built using **HTML5, CSS3, and JavaScript**. This website showcases personal information, technical skills, projects, achievements, and contact details in a professional and visually appealing manner.

---

## 📌 Objective

The objective of this project is to create a fully responsive portfolio website that can be used by students, freelancers, or professionals to showcase their skills, projects, certifications, and achievements online.

---

## 🚀 Features

- 🏠 Responsive Home Section
- 👤 About Me Section
- 🛠 Skills with Animated Progress Bars
- 📂 Projects Showcase
- 🏆 Achievements & Certifications
- 📞 Contact Form (Frontend Validation)
- 🌐 Social Media Links
- 📄 Resume Download Button
- 🌙 Dark / Light Mode Toggle
- ⌨ Typing Text Animation
- ⬆ Scroll-to-Top Button
- 📱 Fully Responsive Design
- 🎨 Modern User Interface
