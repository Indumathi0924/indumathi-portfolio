# Indumathi Portfolio Website

## 📌 Overview
This is a modern, responsive personal portfolio website created by **Indumathi**, a B.Tech student passionate about web development and design.  
The site showcases skills, projects, achievements, and provides contact information in a clean, user-friendly layout.

---

## ✨ Features
- Sidebar navigation for smooth scrolling between sections
- Responsive design for mobile, tablet, and desktop
- Skills displayed in a grid with icons
- Project cards with screenshots and descriptions
- Achievements and certifications section
- Contact form (frontend only)
- Social media links (LinkedIn, GitHub, Twitter)
- Resume download button

---

## 🛠 Technologies Used
- **HTML5** – Semantic structure
- **CSS3** – Styling and responsive layout
- **JavaScript** – Smooth scroll and interactivity
- **Font Awesome** – Icons
- *(Optional)* Bootstrap or Tailwind CSS for advanced styling
